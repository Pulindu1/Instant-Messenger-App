Instructions:
-connect to server
    cd to get to correct directory, then into terminal type:
        python server.py [port]
-connect client to server
    cd to correct directory using a seperate terminal, then type:
        python client.py [username] [hostname] [port]
    click enter 3 times before typing messages!!!!!!
-disconnect client from server:
    type the following disconnect message into chat:
        !DISCONNECT
-unicast messages:
    to unicast messages, send in the form:
        [username] message.....
        e.g. to send hi to username = paul, send [paul] hi
-listfiles
    type following msg to listfiles
        !LISTFILES
-download files:
    type in format:
        !DOWNLOAD [filename]
